# Game Design Document (GDD)
## 가제: Maze Defenders (타워 디펜스 + 덱빌딩 + 미로 빌딩)

---

## 1. 게임 개요

| 항목 | 내용 |
|---|---|
| 가제 | Maze Defenders (변경 예정) |
| 장르 | 타워 디펜스 + 덱빌딩 + 미로 빌딩 |
| 플랫폼 | PC (Windows, Standalone) |
| 시점 | 2D 탑다운 |
| 엔진 | Unity 6.3 (URP) |
| 개발 기간 | 3주 |
| 개발 인원 | 1인 (클라이언트 개발) |
| 타깃 플레이 타임 | 한 판 15~25분 |

## 2. 한 줄 컨셉

> **시작점에서 끝점까지 이어진 타일맵 위에서, 플레이어는 벽을 세워 미로를 만들고, 벽 위에 타워를 짓고, 카드로 얻은 유닛을 타워에 배치해 몰려오는 몬스터를 막아낸다.**

기존 타워 디펜스와의 차별점: **경로 자체를 플레이어가 설계**한다는 것. 기존 TD는 경로가 고정이지만, 이 게임은 벽 설치로 경로가 동적으로 바뀌고 A*가 매번 재계산한다.

---

## 3. 게임 루프

### 3.1 매크로 루프 (한 판)

```
게임 시작
  → 맵 생성 (시작/끝점 결정, 벽 랜덤 배치)
  → 기본 덱·자원 지급
  → [라운드 반복]
       → 빌드 페이즈 (벽 설치, 타워 건설, 유닛 배치, 카드 사용)
       → 전투 페이즈 (몬스터 웨이브 진행)
       → 보상 (자원 + 카드 드래프트)
  → 모든 라운드 클리어 / 끝점 함락
  → 결과 화면
```

### 3.2 마이크로 루프 (1라운드 안)

```
라운드 시작
  → 마나 2 회복, 덱에서 손패 5장 드로우
  → [빌드 페이즈]
       → 자원 카드 사용 → 골드 / 샤드 획득
       → 효과 카드 사용 (마나 소비) → 즉시 효과 발동 (드로우, 마나 획득, 마법 카드 획득 등)
       → 마법 카드 사용 (마나 소비 없음, 1회용) — 빌드/전투 어느 시점이든 가능
       → 유닛 카드를 타워에 드래그&드롭으로 배치 (마나 소비 없음, 카드는 묘지로)
       → 골드로 벽 생성/제거 (벽 위 유닛 배치 = 타워, 별도 타워 건설 비용 없음)
       → 상점에서 카드 구매(골드 + 마나 1)
       → 샤드 교환 (샤드1→골드, 샤드2→마나)
       → 막힘 체크 통과해야 전투 페이즈로 진행 가능
  → [전투 페이즈]
       → 웨이브 진행, 타워·유닛이 자동 공격
       → 적 처치 → 골드 / 샤드 획득
       → 마법 카드 사용 가능 (마나 소비 없음, 1회용)
       → 보스 처치 시 보스 보상(골드/샤드/카드 등) 정산
  → 라운드 종료 → 손패 잔여 카드 + 사용 카드 전부 묘지로
  → 상점 카드 3장 자동 갱신
```

## 4. 핵심 시스템

### 4.1 맵 시스템

- **그리드 기반 타일맵** (확정 크기: **가로 20 × 세로 10**)
- **시작점(Spawn)과 끝점(Goal)** 은 **매판 고정**
  - **시작점: 맵 왼쪽 위 (Top-Left)**
  - **끝점: 맵 오른쪽 아래 (Bottom-Right)**
- **벽 랜덤 생성**: **게임 시작 시 1회만** 실행. 시작점/끝점을 제외한 타일에서 일부를 벽으로 채워 초기 미로 형태 형성
  - **시작점과 끝점 중 적어도 한쪽 인접 타일은 열려 있어야 함** (스폰/목표가 처음부터 막혀 있으면 안 됨)
  - 시작점→끝점 경로가 항상 하나 이상 보장되도록 생성 시 검증
  - 라운드가 바뀌어도 초기 벽은 그대로 유지 (적은 벽을 파괴하지 않으므로 플레이어 조작이 없으면 벽 상태 유지)
- **A\* 길찾기**: 벽이 변화하는 시점에만 경로 재계산 — **빌드 페이즈에서 플레이어가 벽을 설치/제거한 시점에만** 재계산 (전투 페이즈에는 벽 변경이 없으므로 재계산 불필요)

### 4.2 벽 시스템

- 빈 타일에 벽 설치 가능
- **설치/제거 가능 시점: 빌드 페이즈(준비 단계)에서만.** 전투 페이즈 중에는 벽 설치/제거 불가
- **벽 가격**: 자원(골드) 소모
- **벽 파괴 주체: 오직 플레이어만** 벽을 설치하거나 제거할 수 있다. **적은 벽을 파괴하지 않는다.** (전투 페이즈 동안 미로 구조가 변하지 않으므로 A* 재계산 불필요)
- **막힘 처리**
  - 빌드 페이즈 동안에는 자유롭게 설치·제거하면서 미로를 설계
  - 빌드 페이즈 종료 시점에 시작→끝 경로가 막혀 있으면 → 전투 시작 **불가**, 경고 UI로 막힌 구간 하이라이트
  - 플레이어가 벽을 제거해 경로를 다시 열어야 라운드 진행 가능
- **벽 제거 정책** (빌드 페이즈 한정):
  - **설치한 그 라운드 안에 제거 시 → 100% 환불** (즉시 취소 개념)
  - **다음 라운드 이후에 제거 시 → 환불 없음. 오히려 골드를 추가 지불해야 함** (벽 철거 비용)
  - 라운드 경계가 곧 "벽 설치 확정" 시점 — 라운드를 넘긴 미로 변경은 비용이 든다

### 4.3 타워 시스템

**핵심 원칙: 유닛 = 타워.** 벽 위에 유닛 카드를 배치하면 그 자체가 타워가 된다. 별도의 "타워 건설" 단계는 존재하지 않는다.

- **벽 타일 위에만 유닛 배치 가능** (벽 = 타워의 토대)
- 빈 벽 타일은 단순한 장애물 / 적의 경로 차단 역할만 수행 (공격 불가)
- 유닛이 배치된 벽 타일 = 타워 (공격·특수 능력 보유)
- 한 벽 타일에는 **유닛 1명**만 배치 가능 (다중 슬롯 없음)
- 같은 벽 타일에 다른 유닛 카드 드롭 시 교체 (기존 유닛 제거, 새 유닛 배치)
- 타워의 공격력·사거리·특수 능력은 **배치된 유닛 카드의 스탯**으로 결정됨 (별도 타워 등급 없음)

### 4.4 카드 & 덱 시스템

이 게임의 카드 흐름은 **로그라이크 덱빌더 + 마나 비용** 구조다. 한 판 동안만 덱이 유지되며, 게임오버 또는 클리어 시 덱은 리셋된다.

#### 카드 텍스트 표기 컨벤션 — "업그레이드" 지속 시간

카드/보상 설명문에서 "업그레이드"라는 단어의 수식어로 지속 시간이 결정된다. 이 규칙은 모든 카드·보스 보상·효과 텍스트에 일관 적용된다.

| 표기 | 적용 범위 |
|---|---|
| **"공격력 업그레이드"** (수식어 없음) | **해당 판 영구 적용** (게임오버 또는 클리어까지 유지) |
| **"이번 라운드 공격력 업그레이드"** | 해당 라운드 1회 한정 — 라운드 종료 시 해제 |
| **"일시적 공격력 업그레이드"** | 카드/효과에 명시된 짧은 시간(예: n초) 동안만 적용 |

> 디자인 룰: 보스 보상이나 영구 강화 계열은 수식어를 비워 두고, 라운드 한정 마법(준비 페이즈 강화 마법 등)은 반드시 "**이번 라운드**" 접두어를, 전투 페이즈 단시간 버프는 반드시 "**일시적**" 접두어를 카드 텍스트에 포함시킬 것. 컨벤션이 무너지면 플레이어가 강화 지속을 오해해 빌드 의사결정이 망가진다.

#### 카드 종류 (4종)

| 종류 | 사용 방식 | 마나 비용 | 예시 |
|---|---|---|---|
| **유닛** | 손패에서 **마나로는 사용 불가** — 빌드 페이즈에 타워에 드래그&드롭으로 배치 | (배치 자체엔 마나 X) | 궁수, 마법사, 저격수, 지원, 디버프 |
| **효과** | 손패에서 사용 → 즉시 효과 발동 (다양한 보조 효과) | 0~3 | 카드 드로우, 카드 파괴 후 효과 발동, 마나 획득, 유닛 카드 공격력 업그레이드, 광역 데미지, 슬로우, 벽 강화 |
| **자원** | 손패에서 사용 → 골드 / 샤드 획득 | 0 | 골드 +20, 골드 +50, 골드 +100, 샤드 획득 |
| **마법** | **효과 카드의 효과를 통해 획득** → 별도의 마법 탭에 적립 → 게임 플레이 도중 **1회용**으로 사용 | 마나 사용 안 함 | 적 슬로우/스턴(방해형), 단일/광역 공격(공격형) |

#### 마법 카드의 특수 위치

마법 카드는 **덱에 포함된 일반 카드가 아니다.** 효과 카드 중 일부가 "마법 카드 획득" 효과를 가지며, 발동 시 별도의 **마법 탭(인벤토리)** 에 마법 카드가 적립된다.

- **마나를 사용하지 않는다.** 일반 효과 카드와 달리, 마법 카드의 사용 비용은 마나가 아니라 "획득 자체의 희소성"이다.
- **1회용 소모성**: 한 번 사용하면 마법 탭에서 사라진다.
- **라운드 종료 시 누적 유지**: 사용하지 않은 마법 카드는 다음 라운드로 그대로 이월된다.
- **최대 보유 한도: 5장.** 5장이 가득 찬 상태에서 새 마법 카드가 발생하면 **신규 카드는 들어오지 않고 즉시 폐기**된다(기존 5장은 그대로 유지). 교체나 선택지 없음 — 자리를 먼저 비워두는 운영이 필요

#### 마법 카드의 사용 타이밍 (두 갈래 분류)

마법 카드는 **사용 가능한 페이즈에 따라 두 종류로 나뉜다.** 각 마법 카드는 본인이 어느 페이즈에서만 사용 가능한지 고정 속성으로 가진다.

- **준비 페이즈 전용 마법** (빌드 페이즈에서만 사용)
  - **자원 획득형**: 골드/샤드/마나/카드 드로우 등 빌드 페이즈 준비를 돕는 효과
  - **이번 라운드 한정 유닛 강화형**: 이번 전투 라운드 동안 유닛 공격력 업그레이드 등
- **전투 페이즈 전용 마법** (배틀 페이즈에서만 사용)
  - **방해형**: 적 슬로우/스턴/디버프 등 진행 방해
  - **공격형**: 단일/광역 직접 데미지
  - **버프형 (단시간)**: 몇 초간 유닛 타워의 공격속도/공격력 증폭
  - **체력 비율 깎기형**: 보스를 제외한 적들의 현재 체력을 % 비율로 감소

> ⚠️ 보스에는 **체력 % 비율 감소형 마법이 적용되지 않는다.** 보스 처치는 누적 데미지로만 가능하므로 % 깎기형으로 즉사 콤보가 안 됨.

마법 카드 UI에는 사용 가능한 페이즈가 아이콘으로 명시되어야 하며, 현재 페이즈와 맞지 않는 마법은 비활성화 상태로 표시된다.

#### 유닛 카드의 특수 위치 — "방해패" 개념

유닛 카드는 **손패에 들어와도 자원/효과 카드처럼 마나로 발동할 수 없다.** 손패 5장 중 유닛 카드가 차지하는 만큼 자원/효과로 활용할 수 있는 카드 수가 줄어든다. 단, 빌드 페이즈에 손패의 유닛 카드를 타워에 배치하면 유닛이 영구히 그 타워에 배치되고 카드는 묘지로 간다.

즉 유닛 카드는 손패에서 **"이번 라운드 사용 슬롯 손해" + "타워 배치 자산"** 의 양면 가치를 가진다. 이 양면성이 덱 구성의 핵심 트레이드오프다. 유닛 카드가 너무 많은 덱은 자원 빈곤에 시달리고, 너무 적은 덱은 타워가 비어있게 된다.

#### 마나 시스템

- 라운드 시작 시 **마나 2 회복** (기본값, 라운드 진행과 무관하게 고정)
- 효과 카드 사용 시 카드의 마나 비용만큼 소모
- **마법 카드는 마나를 사용하지 않음** (1회용, 마법 탭에서 직접 사용)
- **상점에서 카드 구매 시 마나 1 추가 소모** (가격 무관 고정 1)
- 라운드 종료 시 잔여 마나 소멸 (누적 안 됨)
- 효과 카드의 마나 비용 분포 (권장): 0 마나 30%, 1 마나 40%, 2 마나 20%, 3 마나 10%
- 추가 회복 경로 (라운드 시작 회복 외): **마나 획득 효과 카드**, **마나 보유/획득 마법 카드**, 샤드2 교환
- **후반 라운드 자동 마나 증가는 없음.** 라운드 마나 양은 항상 기본 2이며, 마나 확장은 전적으로 카드(효과/마법) 보유 여부에 달림

#### 드로우 / 묘지 / 덱 사이클

- **라운드 시작 (준비 페이즈 진입)**: 마나 2 회복 + 덱에서 손패 **기본 5장** 드로우
  - 손패 사이즈는 기본 5장이며, **효과 카드 / 일부 상점 카드 효과로 드로우 매수를 늘리거나 줄일 수 있음**
- **최대 손패 상한: 20장.** 드로우 효과로 상한을 넘게 되면, 상한까지만 드로우하고 나머지 카드는 **덱에 그대로 남는다** (묘지로 보내지 않음).
  - 예: 손패 19장 상태에서 "드로우 3장" 효과 발동 → 1장만 손패에 추가, 나머지 2장은 덱 그대로
- **덱이 전부 소모된 상태에서는 드로우 불가.** 덱이 빈 시점에서 시도된 드로우는 그냥 실패하며, 새로 묘지를 셔플해 덱을 재구성하는 동작은 라운드 도중에는 일어나지 않는다 (묘지→덱 복귀는 **다음 준비 페이즈 시작 시점에만** 발생)
- **사용한 카드** (자원·효과 발동, 또는 유닛 카드 벽 위 배치) → 즉시 묘지로 이동
- **사용한 카드는 해당 라운드 안에 다시 등장하지 않음** — 같은 라운드에서 같은 카드를 두 번 쓸 수 없다 (드로우 효과로도 불가)
- **묘지에 있는 카드는 손패로 잡히지 않음** (덱과 별개 영역)
- **라운드 종료** 시 손패에 남아 있던 카드 전부 묘지로
- **다음 준비 페이즈 시작 시 묘지 → 덱 복귀 후 셔플**: 매 라운드 시작마다 묘지에 쌓여 있던 카드 전부가 덱으로 돌아가고 함께 셔플된 뒤 손패를 드로우. 즉 묘지는 **라운드 단위로 비워지는 임시 영역**이며, 라운드 간 덱 재구성은 자동으로 이루어짐 (파괴되지 않는 한 모든 카드가 매 라운드 풀로 복귀)
- **이미 배치된 유닛 카드도 다시 손패에 등장 가능** — 묘지에 들어간 유닛 카드는 일반 카드와 동일하게 다음 라운드 셔플 풀에 포함되며, 다시 드로우되면 새 벽 타일에 또 배치할 수 있음
- **효과 카드의 카드 파괴 효과**: 일부 효과 카드는 손패/덱/묘지의 카드를 "영구 파괴"할 수 있음. 파괴된 카드는 셔플 풀에서 완전히 사라짐 (묘지로 가지 않음)
  - **파괴 대상이 유닛 카드인 경우**: **해당 카드 인스턴스 하나에 대응되는 벽 위 유닛만 제거**됨. 같은 종류(동일 카드명)의 다른 유닛 인스턴스가 맵에 배치되어 있어도 그것들은 영향받지 않음 — 즉 "동일 카드 = 동일 유닛"이 아니라, **카드 한 장 = 유닛 인스턴스 한 개**의 1:1 대응 관계임

#### 시작 덱 (초기 10~12장)

| 카드 | 수량 |
|---|---|
| 골드 +20 (자원) | 4 |
| 궁수 소환 (유닛) | 3 |
| 저비용 효과 카드 (드로우 +2, 광역 데미지 등) | 2~3 |
| 마법사 소환 또는 벽 강화 (보너스) | 1~2 |

#### 상점 (덱 확장 유일 수단)

- 게임 시작 시 상점 매물 **초기 6장** 진열 (기본 매물 슬롯 수: 6)
- **상점 이용 가능 시점: 준비 페이즈에서만.** 전투 페이즈 중에는 구매 불가
- 골드로 구매 → 구매한 카드는 **그 즉시 덱에 영구 추가** (해당 판 끝까지 유지). 다음 라운드부터 드로우 풀에 포함
- **상점 카드 가격은 기본가 고정** (카드 등급별로 정해진 기본가). **효과 카드/특수 효과로 가격이 늘거나 줄 수 있음**
- **구매 시 골드뿐 아니라 마나 1도 함께 소모** (가격이 얼마든 마나 1 고정 소모). 마나가 0이면 구매 불가

**핵심 변경: 매물은 라운드 종료 시 자동 갱신되지 않음.**

- 진열된 6장 중 일부만 구매한 경우, 남은 매물은 **다음 라운드에도 그대로 유지**된다. 예: 6장 중 2장 구매 → 다음 준비 페이즈 상점에는 4장만 진열되어 있음
- **6장을 전부 구매하면 상점이 "품절" 상태**가 됨. 새 매물은 자동으로 채워지지 않으며, 추가 매물이 진열되려면 별도 효과(리롤·매물 확장)가 필요
- 매물을 새로 받으려면 **상점 리롤 카드**(아래 참조) 또는 **매물 확장 카드**(아래 참조)를 사용해야 함

**매물 슬롯 확장 (효과 카드)**

- 효과 카드 중 일부에 **"상점 매물 슬롯 확장"** 효과를 부여할 예정 (디자인 예정)
- 이 효과 카드를 사용하면 상점 매물 슬롯 상한이 6장보다 더 늘어나며, 다음 진열부터 더 많은 카드가 한 번에 진열될 수 있음
- 슬롯 상한 증가는 해당 판 동안 누적 유지

- **무료 드래프트 보상 없음** — 모든 덱 확장은 골드 지출(상점 구매)을 통해서만

#### 상점 리롤

- **골드로 리롤 불가.** 리롤은 카드 효과 / 보상으로 획득한 권리를 통해서만 가능
- 리롤 획득 경로:
  - **효과 카드 중 "상점 리롤" 카드** (디자인 예정) — 사용 즉시 상점 매물 재추첨
  - **보스 보상**의 일부로 리롤권 획득
  - **일부 마법 카드**의 효과 (예: "상점 매물 새로고침" 마법)
- **리롤 시 매물 6장을 새로 뽑되, 기존에 한 번이라도 진열되었던 카드는 후보 풀에서 제외된다.** 즉 이미 본 매물의 재출현을 막아 매번 새로운 선택지가 보이게 함
- 리롤은 준비 페이즈에서만 사용 가능

### 4.5 유닛 시스템

- 유닛은 **유닛 카드를 손패에서 벽 타일에 직접 드래그&드롭** 하여 배치 (별도의 "소환 → 인벤토리" 단계 없음, 별도의 타워 건설 단계도 없음)
- 한 번 배치된 유닛은 영구히 그 벽 타일에서 활동 (해당 판 끝까지 또는 효과로 제거될 때까지)
- 같은 벽 타일에 다른 유닛 카드 드롭 시 교체 (기존 유닛 제거, 새 유닛 배치)
- 카드는 배치 즉시 묘지로 이동 → 셔플 후 다시 손패에 들어올 수 있음
- **이미 배치된 유닛 카드와 동일한 카드가 손패에 다시 들어와도 무방** — 다른 빈 벽 타일에 또 배치 가능 (즉 동일 카드 인스턴스가 맵 위에 여러 개 존재 가능)
- **유닛 카드는 손패에서 마나로 발동 불가** — 오직 벽 타일에 배치하는 방식으로만 사용. 효과 카드/마법 카드로도 활용 불가
- 유닛 속성: 공격력, 사거리, 공격속도, 특수능력
- 유닛 등급 (MVP 5~6종):
  - 궁수형: 단일 공격, 평균
  - 마법사형: 광역 공격, 느림
  - 저격수형: 장거리, 강력, 매우 느림
  - 지원형: 주변 유닛 버프
  - 디버프형: 적 슬로우/약화
  - (선택) 근접형: 짧은 사거리, 강한 공격
- **효과 카드로 유닛 제거 시**: 해당 효과로 카드 자체가 파괴되며, 이미 벽 위에 배치된 유닛이라면 그 유닛 인스턴스도 맵에서 사라짐

### 4.6 자원 시스템

게임은 **마나**, **골드**, **샤드** 세 가지 자원을 분리해서 운용한다.

#### 마나 (카드 사용 자원)

- 라운드 시작 시 **2 회복** (라운드 진행과 무관하게 고정)
- 효과 카드 사용 시 카드의 마나 비용만큼 소모
- **상점에서 카드 구매 시 골드 + 마나 1 추가 소모** (가격 무관 고정 1)
- **마법 카드 사용은 마나를 소모하지 않음** (별도의 1회용 자원으로 취급)
- 라운드 종료 시 소멸 (누적 X)
- 추가 획득 경로: 마나 획득 효과 카드 / 마나 관련 마법 카드 / 샤드2 교환
- **후반 라운드 자동 마나 증가 없음** — 마나 확장은 전적으로 카드 보유로만 가능

#### 골드 (구매 자원)

- **라운드 단위 리셋 자원**: **라운드 종료 시 골드는 0으로 초기화**된다. 즉 골드는 누적 자원이 아니라 **해당 라운드 안에서만 유효한 단기 자원**
- 따라서 라운드 안에서 다 쓰지 않은 골드는 다음 라운드로 이월되지 않음 → 빌드 페이즈 끝나기 전에 골드 소비처를 모두 정리하는 운영이 핵심
- 획득 경로
  - 자원 카드 사용 (골드 +20, +50, +100)
  - 적 처치 보상 (밸런스표 참조)
  - 보스 처치 시 큰 보상(보스 보상)
  - 샤드1 교환 (샤드1 → 골드 1)
- 소비 경로
  - 벽 생성·제거 (설치 라운드 내 제거 100% 환불, 다음 라운드 이후 제거 시 환불 없음 + 철거 비용 추가 지불)
  - **상점에서 카드 구매** (덱 영구 확장 — 유일한 덱 확장 수단, 골드 + 마나 1 필요)
  - (선택) 덱에서 카드 제거
- **주의**: "타워 건설" 별도 비용 없음. 벽 위에 유닛 카드를 배치하면 자동으로 타워 역할 — 골드 추가 소모는 벽 생성 비용으로 갈음됨
- **주의 2**: 상점 리롤은 골드로 불가 (별도 권리 사용)

#### 샤드 (보조 자원)

- **라운드 간 누적 유지 자원**: 골드와 달리 샤드는 **라운드가 끝나도 사라지지 않고 다음 라운드에도 계속 사용 가능**. 라운드 안에서 다 쓰지 못해도 보존되므로 장기 비축 → 후반 폭발용 운영이 가능
- 획득 경로 (주된 출처 순):
  - **자원 카드** (가장 주된 획득 수단)
  - 일부 효과 카드의 부수 효과
  - 보스 처치 시 보스 보상의 일부
- 두 가지 종류로 구분되어 교환 용도로만 사용
  - **샤드1 → 골드 1** 로 교환
  - **샤드2 → 마나 1** 로 교환
- 직접 소비처는 없고, 다른 자원으로 환전해 유연성을 제공하는 "예비 자원" 성격
- **골드는 라운드 종료 시 0으로 소멸**하지만, 샤드를 골드로 환전하면 환전 시점의 라운드에서 사용 가능한 골드가 됨 → 사실상 **샤드 = 라운드 간 가치 저장 수단**으로 기능

### 4.7 준비 페이즈 타이머

준비 페이즈에는 제한 시간이 있다. 시간이 만료되면 자동으로 전투 페이즈로 진입한다.

- **초기 타이머 (1라운드)**: **600초**
- **라운드 진행에 따른 증가**: 다음 준비 페이즈 시작 시 타이머가 **+50초** 추가됨 (즉 2라운드는 650초, 3라운드는 700초 …)
- **타이머 상한: 800초**. 800초에 도달한 이후 라운드부터는 더 이상 증가하지 않음
  - 단순화: 5라운드(600 + 50×4 = 800초)부터 상한 도달
- 타이머는 **준비 페이즈 동안에만 진행**되며 전투 페이즈에는 정지
- 타이머 만료 시: 강제 전투 페이즈 진입. 단 시작→끝 경로가 막혀 있으면 진입 불가(막힘 경고 UI), 플레이어가 경로를 열 때까지 대기
- 준비 페이즈 잔여 시간은 다음 라운드로 이월되지 않음 (라운드 시작 시 새 타이머 값으로 리셋)

> 의도: 초반에는 매우 여유롭게 룰을 익히고 빌드를 짤 수 있도록 600초로 시작. 라운드가 갈수록 +50초씩 늘려 후반의 복잡한 빌드(많은 카드/마법/유닛 운용)에도 충분한 시간 제공. 다만 800초 상한으로 무한정 늘어지는 것은 방지.

### 4.8 적 시스템

- A* 경로로 시작점(왼쪽 위)→끝점(오른쪽 아래) 이동
- **끝점 도달 시 플레이어 체력 감소**:
  - **일반 적**: 플레이어 체력 **1** 감소
  - **보스**: 플레이어 체력 **10** 감소 (한 번만 통과해도 게임이 거의 끝날 수 있는 무게)
- **플레이어 시작 체력: 20** (보스 2회 통과 시 사망 수준)
- **적은 벽을 파괴하지 않는다.** 오직 경로 이동과 본인의 공격 패턴(있다면)만 수행
- 적 종류 (MVP 3종 + 보스):
  - **기본형**: 평균 능력치
  - **고속형**: 빠르고 약함
  - **탱크형**: 느리고 강함
  - **보스**: 라운드 5/10 등장 (끝점 도달 시 -10 데미지)
- 특수형 적(원거리·디버프 면역 등)은 **현 단계에서는 고려하지 않음** — 향후 확장에서 검토
- 웨이브 데이터는 ScriptableObject로 관리

## 5. 메타 진행 (한 판 내부)

- **라운드 사이 흐름**
  - 라운드 종료 직후: 적 처치 보상 + 보스 처치 시 보스 보상 정산
  - **골드 0으로 초기화** (잔여 골드 소멸)
  - **샤드는 그대로 유지** (라운드 간 누적)
  - 묘지 → 덱 복귀 후 셔플 → 손패 5장 드로우 (라운드 시작 시점)
  - 마나 2 회복 (잔여 마나 소멸)
  - **상점 매물은 자동 갱신되지 않음** — 직전 라운드의 미구매 매물이 그대로 남아 있음. 빈 슬롯은 리롤/매물 확장 효과로만 채울 수 있음
  - **준비 페이즈 타이머 +50초 누적** (상한 800초)
  - 다음 라운드 준비 페이즈로 진행 (상점 구매·벽 작업·카드 사용은 준비 페이즈에서만)
- **다음 라운드 시작 시 지급/초기화되는 자원**
  - 골드: **0으로 초기화** (라운드 단위 리셋 자원)
  - 마나: **2 회복** (잔여 마나 소멸, 매 라운드 2로 갱신)
  - 샤드: **이전 라운드 보유분 유지**
  - 시작 손패: **5장** (효과로 증감 가능, 손패 상한 20장)
  - 준비 페이즈 타이머: 이전 라운드 타이머값 + 50초 (최대 800초)
- **무료 카드 드래프트 보상은 없음** — 덱 확장은 오직 골드 지출(상점)을 통해서만

### 5.1 보스 보상 풀

보스 라운드(5/10) 클리어 시 아래 보상 풀에서 하나 또는 여러 종류가 지급된다. 구체적 조합/확률은 밸런싱 단계에서 결정.

| 보상 | 설명 |
|---|---|
| **다음 라운드 추가 손패** | 다음 1라운드 한정으로 시작 드로우 매수가 증가 (예: +2장). 1라운드만 적용되고 그 다음 라운드부터는 기본값(5장)으로 복귀 |
| **랜덤 박스 (카드 1종 획득)** | 카드 풀에서 무작위로 1장 획득해 덱에 영구 추가 (상점 구매와 동일하게 다음 라운드부터 드로우 풀에 포함) |
| **유닛 공격력 업그레이드** | 보유한 특정 유닛(또는 전체)의 공격력을 **해당 판 영구 강화** (텍스트 컨벤션상 수식어 없는 "업그레이드"는 영구 적용 — 4.4 표기 컨벤션 참조) |
| **상점 리롤권** | 상점 매물 6장을 새로 뽑을 수 있는 1회용 토큰 (n회 지급) |
| **자원 보너스** | 골드/샤드 다량 지급 (보조 보상) |

> 비고: 보스 보상은 "다음 한 판을 휘어잡을 변수"를 제공하는 것이 핵심. 누적 보상보다 **즉시 강력함을 체감**할 수 있는 옵션 위주로 구성.

- **(선택) 카드 제거 옵션**: 골드를 내고 덱에서 카드 1장 제거 (덱 비대화 방지)
- **(선택) 메타 영구 진행**: MVP 범위 밖, 향후 확장 — 시작 덱 변경, 해금 카드, 도전 과제 등

### 아키텍처 핵심

- **카드 효과 일반화**: `ICardEffect` 인터페이스를 두고 모든 카드는 데이터 + 효과 리스트로 구성. 효과 추가 = 새 클래스 한 개 작성.
- **이벤트 기반**: `OnWaveStart`, `OnEnemyKilled`, `OnUnitPlaced`, `OnWallPlaced` 같은 이벤트로 시스템 간 결합 최소화.
- **A\* 트리거**: 벽이 추가/제거될 때만 재계산. 매 프레임은 비용 큼.
- **풀링**: 적, 투사체, 이펙트는 Object Pool 필수.

## 12. 결정 사항 & 미결정 사항

### 12.1 결정 사항 (Locked-In Decisions)

플레이 테스트 전 확정된 규칙들.

| 항목 | 결정 |
|---|---|
| 맵 크기 | **가로 20 × 세로 10** (타일맵 그리드) |
| 시작점/끝점 위치 | **매판 고정** — 시작점 왼쪽 위, 끝점 오른쪽 아래. 시작/끝점 인접 타일 중 한 곳은 초기 벽 생성에서 열려 있어야 함 |
| 플레이어 시작 체력 | **20** |
| 적 끝점 도달 시 데미지 | **일반 적 -1**, **보스 -10** (보스 통과는 거의 즉사급 위협) |
| 타워 시스템 | **유닛 = 타워.** 별도 타워 건설 단계 없음. 벽 위에 유닛 카드 배치 → 즉시 타워 역할 |
| 손패 사이즈 | **기본 5장**, 효과 카드/특수 효과로 드로우 매수 증감 가능. **상한 20장** — 초과 드로우 시 상한까지만 채우고 나머지는 덱에 그대로 |
| 덱 소진 시 드로우 | **드로우 불가** (라운드 도중 묘지 → 덱 복귀 없음, 다음 준비 페이즈 시작 시점에만 복귀) |
| 묘지 → 덱 복귀 룰 | **매 준비 페이즈 시작 시 묘지 전체가 덱으로 복귀 후 셔플.** 묘지는 라운드 단위로 비워지는 임시 영역 — 파괴되지 않는 한 모든 카드가 매 라운드 풀로 복귀 |
| 준비 페이즈 타이머 | **초기 600초, 다음 라운드마다 +50초, 상한 800초.** 만료 시 강제 전투 페이즈 진입(경로 막힘 시 진입 불가) |
| 벽·상점 사용 시점 | **준비 페이즈에서만 가능.** 전투 페이즈 중에는 벽 설치/제거·상점 구매 모두 불가 |
| 벽 파괴 주체 | **오직 플레이어만** 벽 설치/제거 가능. **적은 벽을 파괴하지 않음** |
| 라운드당 기본 마나 | **2 마나.** 효과 카드로 라운드 마나 증감 가능 |
| 골드 라운드 처리 | **라운드 종료 시 0으로 초기화** (라운드 단위 리셋 자원, 이월 없음) |
| 샤드 라운드 처리 | **라운드 간 누적 유지** (다음 라운드에도 보유분 그대로 사용 가능) |
| 유닛 카드 손패 운용 | **배치만 가능, 마나로 발동 불가.** 묘지 → 셔플 후 다시 손패에 들어올 수 있고, 이미 배치된 유닛 카드와 동일 카드도 등장 가능(다른 벽에 추가 배치) |
| 상점 카드 가격 | **기본가 고정** (등급별). 효과/특수 효과로 가격이 늘거나 줄 수 있음 |
| 상점 구매 시 자원 | **가격 무관 마나 1 고정 소모** + 골드 |
| 상점 매물 갱신 | **자동 갱신 없음.** 라운드 종료 시 미구매 매물은 그대로 유지, 6장 모두 구매 시 **품절 상태**(새 매물은 리롤/매물 확장으로만 채워짐) |
| 상점 매물 슬롯 수 | **기본 6장**, 효과 카드(매물 확장 카드)로 슬롯 상한 증가 가능 — 해당 판 동안 누적 유지 |
| 상점 리롤 | **골드 리롤 없음.** 효과 카드(리롤 카드) / 보스 보상 리롤권 / 일부 마법 카드로만 가능. **리롤 시 기존에 진열되었던 카드는 후보에서 제외** |
| 카드 제거 옵션 | **효과 카드의 "카드 파괴" 효과로 처리.** 파괴된 카드는 셔플 풀에서 영구 제거. 이미 배치된 유닛 카드가 파괴되면 **해당 인스턴스 1개만** 맵에서 제거 — 같은 종류 다른 유닛 인스턴스는 유지 |
| 마법 카드 운용 | **라운드 종료 시 누적, 최대 보유 5장**, 1회용 사용 |
| 마법 카드 5장 초과 시 | **신규 카드 못 들어옴 (자동 폐기)**, 기존 5장은 그대로 유지 |
| 마법 카드 사용 페이즈 | **각 마법은 "준비 페이즈 전용" 또는 "전투 페이즈 전용" 속성을 가짐** — 모든 페이즈에서 자유롭게 쓸 수 없음. 준비용은 자원/드로우/이번라운드 강화, 전투용은 방해/공격/단시간 버프/체력 % 깎기 |
| 보스 면역 | **체력 % 비율 감소형 마법은 보스에게 적용되지 않음** |
| 보스 보상 풀 | **다음 라운드 추가 손패(1라운드 한정) / 랜덤 박스(카드 1종 획득) / 유닛 공격력 업그레이드(영구) / 상점 리롤권 / 자원 보너스** 중 일부 조합 지급 |
| 샤드 획득 경로 | **주로 자원 카드,** 일부 효과 카드 + 보스 보상 |
| 마나 획득 효과 카드 | 효과 카드 카테고리 내 별도 카드로 디자인 예정 |
| 후반 라운드 마나 증가 | **자동 증가 없음.** 마나 확장은 카드(효과/마법) 보유로만 가능 |
| 적 종류 | **벽 파괴자·특수형 제외**, 일반 적 3종(기본/고속/탱크) + 보스 |
| 벽 제거 환불 | **설치한 라운드 내 제거 시 100% 환불**, 다음 라운드 이후 제거 시 환불 없음 + 철거 비용 골드 추가 지불 |

### 12.2 미결정 사항 (Open Questions)

플레이 테스트와 프로토타이핑을 통해 결정해야 할 항목들.

1. **다음 라운드 이후 벽 철거 비용** — 환불 없음 + 추가 골드 지불 룰은 확정. 철거 비용 금액(설치가의 50%? 100%? 고정값?)은 밸런싱으로 결정
2. **샤드1/샤드2 드롭 비율** — 자원 카드 디자인 시 균형 결정 필요
3. **보스 보상 구성** — 5종 풀(추가패/랜덤박스/공격력업/리롤권/자원보너스) 중 한 보스당 몇 가지 옵션 노출, 어떤 비율로 묶을지
4. **리롤 카드/매물 확장 카드의 등급·가격·드롭률** — 상점 매물이 자동 갱신되지 않으므로 이 두 카드의 수급이 너무 적으면 후반 덱 확장이 막힘. 풀 내 비중 조정 필요
5. **상점 구매 시 마나 1 소모 룰의 영향** — 라운드 마나가 2뿐이므로 상점 구매가 효과 카드 사용을 강하게 견제함. 의도된 트레이드오프인지 플레이 테스트로 검증 필요
6. **마나 획득 카드의 비중** — 효과/마법 카드 중 마나 회복 계열의 적정 비율. 너무 많으면 기본 2 마나 제약이 무의미, 너무 적으면 카드 사용이 마름
7. **골드 라운드 종료 0 초기화 룰의 강도** — 골드가 단기 자원이 되면서 "샤드 비축 후 환전" 전략이 핵심이 됨. 자원 카드의 골드/샤드 분배가 너무 골드 쪽이면 후반에 골드 낭비가 심해지고, 너무 샤드 쪽이면 즉시 구매가 어려워짐 — 밸런스표에서 분배 비율을 시험해야 함
8. **준비 페이즈 타이머 600/+50/800 곡선이 실제 플레이 부담에 맞는지** — 초반 600초는 길고 후반은 부족할 수 있음. 플레이 테스트 후 시작값/증가폭 조정
9. **품절된 상점에서 리롤·매물 확장 카드 수급이 막혔을 때의 대처** — 극단적 케이스에서 상점이 완전히 비고 리롤 카드도 손에 없어지면 덱 확장이 불가. 안전망(예: n라운드마다 강제 매물 1장 보충)이 필요한지 검증 필요
